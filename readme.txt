== Description ==

This plugin allows you to add a widget on your website which shows the business hours of your business on your website


== Installation ==

Here is what you need to do: 

1. Install plugin OR unzip the downloaded file to /wp-content/plugins/
1. Go to your plugin settings and enable the plugin.
1. In the sidebar you will have a new section called 'Contact Us' navigate there and set your preferences.
1. goto appearance = > widget,you will see a plugin named "Lordlinus buisness hours", activate it and you will be able to see your buienss hour when updated through the plugin
